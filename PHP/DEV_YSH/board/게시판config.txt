CRUD

1. C : 글 작성
2. R : 글 읽기
3. U : 글 수정
4. D : 글 삭제

-------------------------------------------------

1. 글 읽기 : list : 데이터베이스에 있는 row를 가져와서 view 해주는 기능
2. 글 작성 : regist : 사용자가 입력한 데이터를 데이터베이스 Colmn에 insert 하는 기능
3. 글 수정 : update : 데이터베이스에 있는 row를 사용자가 조건을 충족하였을 때, uid를 기준으로 해당 내용을 update하는 기능
4. 글 삭제 : delete  : 데이터베이스에 있는 row를 사용자가 조건을 충족하였을 때, uid를 기준으로 해당 내용을 delete하는 기능
5. paging 기능
